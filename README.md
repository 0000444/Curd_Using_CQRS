
# CRUD Using CQRS and MediatR

A clean .NET 8 Web API demonstrating CRUD operations with CQRS pattern and MediatR library.

## 🚀 Tech Stack

- .NET 8 | Entity Framework Core | MySQL
- CQRS Pattern with MediatR
- ASP.NET Core Identity
- Swagger & Scalar API Docs

## 📡 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/Auth/register` | Register user |
| GET | `/api/Blog` | Get all blogs |
| GET | `/api/Blog/{id}` | Get blog by ID |
| POST | `/api/Blog` | Create blog |
| PUT | `/api/Blog/{id}` | Update blog |
| DELETE | `/api/Blog/{id}` | Delete blog |
